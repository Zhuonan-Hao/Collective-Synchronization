# Sworm Synchronization

Research project about multiple agents synchronization behavior

## Installation

OS X & Linux:

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pybullet.

```bash
pip install pybullet
```

## Usage

```python
python sworm.py
```
